Instuções Fisebase:

(O símbolo ">" é somente uma representação da execução da consola. Caso queira utilizar os comandos mencionados abaixo, escreva somente o que se encontra em frente deste mesmo símbolo)


Comando para instalar as ferramentas do Firebase:

	>-npm install -g firebase-tools


Comando para associar a sua conta Firebase com as ferramentas do Firebase instaladas anteriormente:	
	
	>firebase login
	
	-Irá ser aberto um separador no seu browser predefinido onde deverá efetuar login com a sua conta.


Comando para iniciar o Firebase pelo cmd/Powershell Integrated Console:

	>firebase init

	-Irão surgir vários pedidos, basta selecionar o que pretende. Neste caso, foram implementadas as funções Database, Firestore e Hosting.


Comando para executar o website localmente:

	>firebase serve


Comando para implementar alterações para o diretório alojado no Firebase:

	>firebase deploy
